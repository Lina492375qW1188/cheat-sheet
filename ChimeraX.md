Orthographic view
```
camera ortho
```

Delete H-bonds
```
hbonds delete
```

Remove everything except "C"
```
hide ~@C
show @C
```

Show coordinate tripods
```
cofr showPivot true
```

Set position of coordinate tripods
```
cofr fixed -5, -5, 0
```

Customize length and radius of axes
`cofr showPivot 5, .25`
